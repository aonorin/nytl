Math

Detailed Description

The math module deals with utility template classes and functions. Especially the vec class is designed with glsl in mind. It does additionally include all sort of classes useful for graphical computing. There are some specific design patterns, so are most of the operations or tests you can do with a class defined as global functions instead of member functions.
