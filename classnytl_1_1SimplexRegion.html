Detailed Description

template<std::size_t D, typename P = float, std::size_t A = D>
class nytl::SimplexRegion< D, P, A >

Describes a RectRegion of multiple Simplexes.

Member Function Documentation

Adds a Simplex to this RectRegion.

void nytl::SimplexRegion< D, P, A >::subtract ( const SimplexType &  simplex)

Subtracts a Simplex from this SimplexRegion.

Effectively checks every Simplex of this SimplexRegio for intersection and resizes it if needed.
