Detailed Description

template<typename Key, typename Base = cache>
class nytl::multiCache< Key, Base >

Base class for classes that carry associated cache objects.

Objects of classes dervied from multiCache are able to hold multiple associated cache Copying one multiCache object does clone all its cache objects, it may be therefore an expensive operation. If the multiCache object is move-constructed or move-assigned no cache object has usually to be moved or copied.
