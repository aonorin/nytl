Returns the gradient vector in relation to the given dimension parameter. If e.g. dim is 0, the x component of the returned gradient vector will be 1 and all other components will be set in relation.

template<size_t D, typename P>
vec_type nytl::simplex< D, P, 1 >::valueAt ( const value_type &  value,
std::size_t  dimension = 0 
) const

Returns the point of the line at the given value in the given dimension. One should check with definedAt(value, dimension) if the line is defined for the given value before using this. If it is not, this function will produce a warning and return an empty vec.
