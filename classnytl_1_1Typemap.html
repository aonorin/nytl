Detailed Description

template<typename I, typename B = Any, typename... CArgs>
class nytl::Typemap< I, B, CArgs >

Can be thought of as map (like std::map) that holds pair of identifiers and types.

Internally just uses type erasure to "store" its types. Therefore there are just a small set of operations that can then be done with those "stored" types, like e.g. creating them from identifier or std::type_info or receive the type_info of the stored type by identifier.

Template Parameters
IIdentifier type for the types (usually std::string)
BBase type of all stored types. If there is none, use void (Created objects will then be passed as void* objects). Defaulted to void
CAAdditional construction args an objects needs on creation.
